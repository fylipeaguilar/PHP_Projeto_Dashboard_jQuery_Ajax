# PHP_Projeto_Dashboard_jQuery_Ajax

Esse projeto foi desenvolvido com os estudos da plataforma Udemy.

Dentre o projeto, uma parte foi realizada junto com as aulas e as demais partes eu desenvolvi sozinho.
